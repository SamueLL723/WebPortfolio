This is the start of my journey into web development. I need to build a functional website for a school project. I wasn’t really a fan of what we were doing in school, so I decided that I want to do it on my own and in my own way.

At some point I told myself that I’d create my own web portfolio where I can train and try things out. And once I get the hang of it, I’ll make the actual school project — a website called RacePredictorGP. So basically, this page is just for testing and learning.

I have no idea how to actually build it, but with the help of AI I got a rough idea of what tools I want to use. The only problem is that I’ve never used some of the tools it recommended to me. So I asked it how to start. It gave me a daily plan of what I need to do so that I’ll be able to build a functional website — a 13-day plan. So let’s see how it goes and how well I’ll do.
